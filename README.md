# Basic Map with the ArcGIS API 4.0 Beta 2 for JavaScript
A sample app that demonstrates some of the additions to the ArcGIS API for JavaScript at version 4.0 Beta 2:
- [Popups](https://developers.arcgis.com/javascript/beta/api-reference/esri-widgets-Popup.html)
- [Geoprocessing](https://developers.arcgis.com/javascript/beta/api-reference/esri-tasks-Geoprocessor.html)
- [IdentityManager](https://developers.arcgis.com/javascript/beta/api-reference/esri-identity-IdentityManager.html)
- [Locate widget](https://developers.arcgis.com/javascript/beta/api-reference/esri-widgets-Locate.html)
- [Vector tile layers](https://developers.arcgis.com/javascript/beta/api-reference/esri-layers-VectorTileLayer.html)

[View the web app now](https://cdn.rawgit.com/garys-esri/basic-map-js-4.0-beta-2/v1.0.0/site/index.html)
